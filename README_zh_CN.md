# zh-CN

超融合(Super Polymerization)主题是融合了多个优秀样式风格的主题外观，有很好的视觉效果，尽情享受视觉盛宴吧！主要分为两个主题，一个是白天使用的元素使者，一个是晚上使用的暗夜使者。

## 目录结构
```
.__dart 黑夜模式依赖文件
|__tiangesec 白天主题
|__tiangesecdark 黑夜主题
```
